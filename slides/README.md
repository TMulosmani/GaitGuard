# TorqIt Frontend

React frontend for the Knee Torque Tracker system.

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Development

The dev server runs on `http://localhost:5173` by default.

## Build

```bash
npm run build
```
